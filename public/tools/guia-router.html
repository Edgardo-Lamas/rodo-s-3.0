Internet Segura para la Familia

Guía de configuración del router · Referencia interna de instalación

¿Para qué sirve configurar el router? Al configurar el DNS en el router, todos los dispositivos del hogar quedan protegidos automáticamente (Smart TVs, consolas, notebooks, celulares), sin necesidad de configurar cada uno por separado.

Servidores DNS a configurar en todos los casos:

DNS Primario: 1.1.1.3 DNS Secundario: 1.0.0.3

Cloudflare for Families — bloquea adultos, malware y phishing.

TP-Link Archer / TL-WR series
1

Ingresar a 192.168.0.1 desde cualquier navegador.

2

Usuario: admin · Contraseña: admin (o la personalizada del cliente).

3

Ir a Avanzado → Red → DHCP.

4

En los campos DNS Primario y DNS Secundario ingresar los valores indicados arriba.

5

Guardar y reiniciar el router.

Fibertel / Flow Routers de Telecom / Cablevision
1

Ingresar a 192.168.0.1 o 192.168.100.1 según el modelo.

2

Usuario y contraseña están en la etiqueta del router (generalmente admin / admin o user / user).

3

Ir a Configuración LAN → Servidor DHCP.

4

Completar los campos de DNS con los valores indicados.

5

Guardar y reconectar los dispositivos del cliente al WiFi.

En algunos modelos de Fibertel el DNS está en Avanzado → DNS directamente.
Claro / Personal Routers de AMX / Telecom
1

Ingresar a 192.168.1.1.

2

Credenciales en etiqueta del equipo. Comunes: admin / 1234 o admin / admin.

3

Ir a Red Local → DHCP → Opciones avanzadas.

4

Completar DNS Primario y Secundario.

5

Aplicar cambios y reiniciar.

Cualquier otro router Procedimiento genérico
1

La IP del router suele ser 192.168.0.1 o 192.168.1.1. Se puede verificar con ipconfig en Windows → ver "Puerta de enlace predeterminada".

2

Buscar la sección de configuración DHCP o LAN.

3

Localizar los campos DNS y completar con los valores indicados al inicio de esta guía.

4

Guardar y reiniciar el router.

Resolución de problemas comunes
No puedo acceder al panel del router

Probar las dos IPs habituales: 192.168.0.1 y 192.168.1.1. Si ninguna funciona, ejecutar ipconfig en Windows y buscar "Puerta de enlace predeterminada".

Guardé los DNS pero los dispositivos siguen sin filtrado

Desconectar y reconectar cada dispositivo al WiFi para que tome la nueva configuración DHCP. También limpiar el cache DNS en Windows con ipconfig /flushdns.

El router no deja modificar el DNS

Algunos routers de ISP tienen el panel bloqueado. En ese caso configurar el DNS en cada dispositivo individualmente (script Windows + perfil iOS + guía Android).
